HTML 기본 규칙
  • 특수문자는 Entity Code를 사용한다.
  • label이 없는 input, select는 title 속성을 넣는다.
CSS 기본 규칙
  • 모든 속성은 영문 소문자로만 작성하며 카멜 기법을 사용한다.
  • @charset 선언시에만 큰따옴표("")를 사용하고 그외의 속성 파라미터값에는 작은 따옴표('')를 사용한다.
  • background의 url 속성의 파라미터값에는 따옴표를 사용하지 않는다.
  • 마지막 속성의 세미콜론(;)은 삭제한다.
  • css 코드는 작성할때는 들여쓰기를 하지 않는다.
  • 속성간에는 공백, 줄내림을 하지 않으며 모든 속성을 붙여쓰기 한다. 단, scrollbars, CSS3에서 배경그라데이션 색상과 같이 하나의 속성을 여러개의 핵을 써야할 경우에는 허용한다.
  • 중괄호 좌우 공백은 제거한다.
  • 의미있는 객체를 구분하기 위한 빈 줄은 한줄을 초과하지 않게 한다.
  • 약식 속성을 이용하되 속성값이 하나만 있는 경우는 약식 속성을 이용하지 않는다.
  • 최적화를 위해 '0'뒤의 단위는 생략한다.
  • 반복되는 색상값은 축약해서 사용한다.(#555555 -> #555)
CSS 속성 선언 순서
박스모델을 기준으로 바깥에서 안쪽 순서대로 선언한다.
  • display, zoom
  • overflow
  • float
  • position, top, left, right, bottom, z-index
  • margin, border, padding
  • width, height
  • text-align, vertical-align, text-decoration, text-indent
  • font, letter-spacing, white-space, color
  • background
  • 기타(위에 언급하지 않은 나머지 속성들)
